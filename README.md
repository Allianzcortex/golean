# golean

#### 这是什么 ?

一个将视频截图综合在一起只保留字幕的程序

支持自定义截屏字幕的高度

#### 使用方法


下载 bin 目录中的文件，提前编辑好配置文件，在 `config.toml` 中设置选项，之后执行`./process_image`(Linux) 或者 `./process_image.exe`(Windows)
